# DecodeLabs-Internship
A simple to do list that operates directly within your terminal. Just Python without any installations or additional libraries.

**Purpose of Project :-**
Easily keep track of your everyday tasks;

Choices you can use are:- 
1. Add tasks 
2. View all tasks 
3. Mark tasks as completed
4. Remove tasks

Software Used :-
Python 3.x


When you launch the application, a menu shows up:--- To-Do List ---
1. Include a task
2. View the task
3. Mark the task as completed.
4. Remove the Task
5. Exit

Simply input a number and proceed!
